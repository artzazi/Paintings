ZAZAA GANBOLD WEBSITE V2

This version includes working language switching for English, German and Mongolian.

Upload to Netlify

1. Unzip this ZIP file.
2. Open Netlify.
3. Choose Add new site.
4. Choose Deploy manually.
5. Drag the whole unzipped folder into Netlify.

How to edit texts

Open the folder data.

en.json controls English.
de.json controls German.
mn.json controls Mongolian.

How to edit artworks

In each language file, find the works section.
Change title, medium, size, status or series text there.

How to replace images

Open assets/images.
Replace an image with a new file that has the same filename.
